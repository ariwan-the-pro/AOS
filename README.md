How to set up AOS

1. Go to cmd

2. Type wsl --install

3. Do the Linux setup.

4. When you are in WSL, type mkdir aos

5. Then cd aos

6. Type nano aos.sh

7. Type all the code from the GitHub page into the file

8. Ctrl+O, enter, Ctrl+X

9. If you now type ./aos.sh OR bash aos.sh it will take you to the OS

10. YOU ARE DONE!
